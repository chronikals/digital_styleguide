Links

Links to other pages or external references should be clearly visible and unambiguous given the context. When using inline text links in body text, the text color and weight should clearly distinguish from body text. Consider using underlining, depending on platform. In separate link sections or in bread crumbs the context provides more guidance, so less link styling is required.
