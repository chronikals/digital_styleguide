Color themes

Color themes can be used to adjust an individual application to different circumstances - such as shifting daylight throughout the day - or to allow users to adapt the user interface according to personal preferences. In any case, it is important to ensure that the color philosophy is maintained regardless of theme chosen, so that the visual layering of information is preserved.

Light and dark color palettes

Maintaining visual hierarchy in data visualizations

When designing light and dark palettes, it is important to pay particular attention to data visualizations. Maintaining the visual hierarchy so the most important information stands out, while background data and context is allowed to take a back seat should be achieved on both palettes.

Adjusting colors

As colors appear differently on different backgrounds, the actual colors used for foreground elements must be adjusted individually. This is not a strictly mathematical operation, but requires visual tuning and adaptation.
