Normal button

This is the default button that is typically used for actions and initiating dialogs that do not require any particular emphasis.

Primary action button

A primary action button can be used for any call-to-action or step that requires specific emphasis. Typically to commit an action or to proceed to next step.

Sizes

In this version of the style guide, only interface components for desktop applications (web and native applications) are specified in pixels. Guidelines for touch interfaces and other hardware formats will follow in later versions of the style guide.

Height

The height of a button is defined by the font size. For a normal desktop application the default font size is 12 px and the default button height is 36 px (3 x font size).

If the default button is too big for your interface, smaller buttons can be used. Just make sure that they align to the baseline and that your buttons are at least twice the font size.

Width and padding

The length of your button label and the padding define the width of your button.

The padding is equal to the font size, but for the minimum button the padding can be decreased to half the font size (ex. font size 12 px, padding 6 px).

Text only button

Text is by default centered on buttons.

Text and icon

Icon and text is by default left aligned, but can be adapted based on context.
