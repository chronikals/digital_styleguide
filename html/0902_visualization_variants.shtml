Pie charts (Part to whole)

Considerations when using Pie charts:

  • Pie charts are generally over-utilized; bar charts and column charts allow the users to more easily see the relations between the data. It is much more difficult to visually judge the size of circles (or circle segments) vs. rectangles.
  • If using pie charts, the number of data points (segments) should be low (3-5)
  • The underlying data should add up to 100% (which may necessitate the inclusion of a category such as “none”, “other”, etc.)
