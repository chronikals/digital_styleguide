Data visualization

Visualizations are used to make data interpretable and easier to use as information. There are numerous types of visualizations available, and the best suitable option will depend on context, type of data and what the user is trying to achieve:

  • Does the user need to quickly detect outliers in the dataset?
  • Does the user need to compare data in time or between categories?
  • Is the user trying to understand the composition or distribution of the data?

This chapter starts by providing some general recommendations for data visualization design, and then provides a collection of typically used charts intended for inspiration.
