CommonUX Design principles

Clarity - Workflow - Focus

  • Give clarity to complex information through a simple and rational framework
  • Support user work flow by offering easy to understand components and navigation
  • Focus on clear presentation of dynamic information.

ABB UI style guide

The styleguide is eintended to be an overarching, top-level guide valid for all SW products in all divisions in ABB. It is intended to provide the basic level of harmonization between ABB products.

Product or BU-level style guides (optional)

Some BUs and products might choose to develop a more detailed style guide to create a closer alignment with less variability between products or screens, within the boundaries set by the ABB UI style guide. Typically this can specify further a limited set of layout options, narrow down the choice of colors used and specify interaction patterns.

Product specifications

Typically, the handover from interaction designers to the product developers will still be the detailed, screen by screen specifications necessary to develop the product. By applying the style guides above the specification process will be more efficient as much will already be defined above, and similar from product to product.
