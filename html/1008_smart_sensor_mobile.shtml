Smart sensor mobile Mobile application

The ABB Smart sensor is a pocket-sized sensor that can be retrofitted to existing low-voltage motors without the need for any infrastructure or wiring. The Smart sensor mobile applications communicates directly with the sensor via Bluetooth, and is used to configure new sensors and read/synchronize sensor data.

More info: http://new.abb.com/abb-ability/smart-sensors
